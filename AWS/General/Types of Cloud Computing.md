- [[AWS]]
- Infrastructure as a Service (IaaS)  **Amazon EC2**
	- Provide building blocks for cloud IT
	- Proved networking, computers, data storage space
	- Highest level of flexibility
	- Easy parallel with traditional on-premises IT
- Platform as a Service (PaaS)  **Elastic Beanstalk**
	- Removes the need for you organization to manage the underlying infrastructure
	- **focus on deployment and management of your applications**
- Software as a Service (SaaS) 
	- Completed product that is run and managed by the service provider

# Pricing of the Cloud
- AWS 3 pricing fundamentals, pay-as-you-go model
- Compute
	- pay for compute time
- Storage
	- Pay for data stored in the cloud
- Data transfer OUT of the Cloud
	- Data transfer IN is free
- **Solves the expensive issue**
